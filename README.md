# gamemt-autoinstaller
Plug &amp; Play configuration for GameMT E6 Plus and Max
# Install
Paste this command in termux: 
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Saafork/gamemt-autoinstaller/refs/heads/main/setup.sh)"`

# Doesn't configure Retroarch
This tool **DOWNLADS** retroarch. It does not configure it.
